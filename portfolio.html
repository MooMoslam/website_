Master circuit simulation

"We used Proteus to simulate the Arduino Mega and test sensor readings before implementing the hardware. This allowed us to validate the system’s behavior, fine-tune configurations, and ensure accurate data acquisition. By identifying and addressing potential issues early, the simulation helped us save development time and minimize hardware-related errors."

ESP 32

Web Server

"In this system, the ESP32 functions as a slave device and is configured to operate as a standalone Access Point. It establishes a wireless interface through which it communicates with the Arduino Mega, receiving live sensor data and sending back control commands. These include robot movement directions, disinfection process activation, and emergency stop signals. Additionally, the ESP32 hosts a dynamic web interface that allows users to monitor sensor readings and control system operations in real-time through any connected device."

Autonomous

Sterilization Robot

"Final Outcome of the Robot"
The robot utilizes a combination of advanced tools and sensors to achieve seamless operation and efficient disinfection. The key components include:
📌The robot integrates a UV lamp for disinfection
📌Spray system for misting disinfectant
📌3 Ultrasonic sensors for obstacle detection
📌DHT22 sensor to monitor room conditions
📌DHT11 sensor to track internal temperature and humidity
📌Water level sensor for the tank
📌Battery sensor for power monitoring
📌Motors with their drivers for precise movement
These tools were carefully selected, and the necessary calibrations were carried out to obtain the highest and most accurate readings from these sensors and avoid any errors.
This enabled us to create a scalable, cost-effective, and highly efficient disinfection robot that is competitive in the market.
